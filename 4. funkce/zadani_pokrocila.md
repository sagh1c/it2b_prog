
# Další úlohy

## 1. Papoušek
Vytvoř funkci `papousek(char text[])`, která vypíše text dvakrát za sebou.

## 2. Opakování textu
Vytvoř funkci `opakuj(char text[], int pocet)`, která vypíše text zadaný počet-krát.

## 3. Tisk řádku
Napiš funkci `tiskRadek(char znak, int delka)`, která vypíše na jeden řádek znak opakovaný `delka`-krát.

Např.: `tiskRadek('#', 5)` → `#####`

## 4. Tisk pyramidy
Napiš funkci `tiskPyramida(char znak, int vyska)`, která vypíše pyramidu dané výšky.

Např. pro `tiskPyramida('o', 3)`

```
o
oo
ooo
```

## 5. Porovnání dvou stringů
Vytvoř funkci zjišťující, zda jsou dva stringy stejné (1 = stejné, 0 = rozdílné). Nepoužívej `strcmp()`.

## 6. Validace PINu
Vytvoř funkci `validujPin(int pin)`. PIN je platný, pokud je čtyřmístný a dělitelný čtyřkou.

## 7. Validace data narození
Vytvoř funkci `validujDatumNarozeni(int rok, int mesic, int den)`, která ověří, zda je datum validní.
Rok menší než 1900 není platný.
